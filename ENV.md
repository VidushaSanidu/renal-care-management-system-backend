# Backend Environment Variables

> NOTE: The .env.example file provides example values for all environment variables. You must create a .env file with appropriate values before running the backend server. Please read the explanations below for each variable to understand how to configure them properly.

## JWT Configuration (must match the ML server)

This setting defines the secret key used for signing JSON Web Tokens (JWTs) for authentication. It must be the same as the `JWT_SECRET` used in the ML server to ensure that tokens generated by the backend can be verified by the ML server. Change this to a secure value before deploying to production. also make sure to set `JWT_EXPIRE` to an appropriate value (e.g., `1d` for 1 day) to control how long the tokens are valid.

There is no default value, so you must set it before running the backend server.

```env
JWT_SECRET=your-super-secret-jwt-key-here-change-in-production
JWT_EXPIRE=7d
```

## Express.js Configuration

`BACKEND_PORT` is set to `5000` by default, but you can change it to any available port if needed. The `NODE_ENV` variable should be set to `production` in production environments for better security.

```env
BACKEND_PORT=5000
NODE_ENV=development
```

## Database Configuration

This backend uses MongoDB as the database. You need to provide the connection URI for your MongoDB instance. If you are running MongoDB locally with default settings, you can use the following URI. For production, you should use a secure connection string that includes authentication credentials.

```env
MONGODB_URI=mongodb://localhost:27017/renal-care-db
```

## Other Configuration

This setting defines the URL for the ML server that the backend will communicate with. Make sure to set this to the correct URL where your ML server is running.

```env
ML_SERVER_URL=http://localhost:8001
```

This setting defines the URL for the React frontend that the backend will communicate with. Make sure to set this to the correct URL where your React frontend is running.

```env
FRONTEND_URL=http://localhost:3000
```
